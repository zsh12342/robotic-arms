# TrajectoryPlanner
A trajectory planner via moveIt

```bash

├── config    # moveit配置文件
├── example   # 示例程序（里面有readme可查看）
├── traj      # 生成好的轨迹
├── __init__.py  # 初始化 
├── base.py      # 控制基础类
├── calibrator.py  # 校准器
├── exception.py   # 异常类
├── executor.py    # 执行器
├── logger.py      # 日志类
├── optimizer.py   # 优化器
├── planner.py     # 规划器
├── publisher.py   # 发布器
├── scene.py       # 场景类
├── transformer.py # 变换器
├── utils.py       # 工具类
└── README.md      # 说明文档
```
